# Oregon Trail-Inspired Python Game

A command-line Python game inspired by Oregon Trail, built to practice programming fundamentals such as file handling, save/load functionality, user input handling, random events, and state management.

## Features
- Command-line gameplay
- Inventory and resource tracking
- Random events and encounters
- Save and load functionality
- Turn-based progress and status updates

## Tools Used
- Python

## What I Practiced
- Program flow and logic
- File handling
- State management
- Debugging and iterative testing
- User input handling

## How to Run
1. Make sure Python is installed
2. Download the project files
3. Run the program from the terminal:

```bash
python main.py
